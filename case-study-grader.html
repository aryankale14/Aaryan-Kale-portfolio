AI-Based Assignment Grader & RAG Assistant

Problem

Manual evaluation of handwritten exam papers is slow, inconsistent, and does not scale.

Solution

Built an AI-driven grading system using AWS Textract for OCR and Gemini for evaluation, combined with a RAG-based assistant for syllabus queries.

Architecture

  • Frontend: Web Interface
  • Backend: Flask API
  • OCR: AWS Textract
  • LLM: Gemini
  • Vector DB: Pinecone

Impact

Achieved ~92% grading accuracy and reduced evaluation time by over 70%.
